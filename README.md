# assistive-tech-api
An API with Assistive Technology Lending Libraries in the U.S. Updated June 2022.


![](AssistiveTechLibrary.png)

## :grey_question: How to use it?

#### To view all states
```sh
https://assistive-tech-library-api.herokuapp.com/api/lending-libraries
```

#### To view specific state
```sh
https://assistive-tech-library-api.herokuapp.com/api/lending-libraries/{state name capitalized}
```
