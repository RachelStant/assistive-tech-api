Assistive Technology Lending Library API

This API provides a list of assistive technology & augmentative and alternative communication lending libraries across the US. You can view all libraries at once, or select them by state. (Information subject to change. Updated June 2022.)

All States:

https://assistive-tech-library-api.herokuapp.com/api/lending-libraries

Specific State (capitalize state name ex. Pennsylvania):

https://assistive-tech-library-api.herokuapp.com/api/lending-libraries/state
